# 🪨 Rock Paper Scissors Game 🎮

This is a simple **Rock, Paper, Scissors** game written in Python.  
You can play it directly in your terminal by choosing between:
- `r` → Rock 🪨
- `p` → Paper 📃
- `s` → Scissors ✂️
